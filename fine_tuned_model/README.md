# Fine-tuned model (local only)

Model weights are not committed because they exceed GitHub's file size limit.

Expected directory contents for running the apps:

- `fine_tuned_model/config.json`
- `fine_tuned_model/tokenizer.json` (and other tokenizer/config files)
- `fine_tuned_model/model.safetensors` (weights)

If you trained the model locally, copy/export the weights into this folder before running the apps.
